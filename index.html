    • Introduction

    JavaScript is a cross-platform object-oriented scripting language.

    It contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements

  • Client side extends the core language by supplying objects to control browser and DOM. For example, it allows an application to place elements on an HTML form and respond to user events such as mouse clicks, form input, and page navigation
  • Server-side extends the core language by supplying objects relevant to running JavaScript on a server. It allows an application to communicate a database, provide continuity of information from one invocation to another of the application, or perform file manipulations on a server

    Variables

    Variables are like symbolic names in your application. The name of variables, called identifiers, conform to certain rules

    var x =42 let y = 13 if (true) { let y=5; } console.log(y); if (true) { let x=5; } console.log(x);
